# Mistery-Box-PT-BR
Mystery Box é um jogo interativo de investigação baseado em lógica, apresentado como uma coleção de arquivos abandonados.

No Case 01, o jogador acessa o site aparentemente inocente de uma doceria. Explorando seções como menu, ingredientes, clientes, comentários e notícias, é preciso conectar pistas sutis para descobrir o que realmente aconteceu.
